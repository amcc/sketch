Ortho

Ortho is a 3D grid of orthographically displayed boxes. It is an experiment with code to create an infinitely variable composition with form and colour.

Use the controls to alter the artwork, each control has a keyboard shortcut, on a mobile you can tap the buttons instead.

By saving the artwork you download a completely unique image, timestamped with a unique name.
